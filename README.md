# Samuele Felici

Personal landing page — [samuelefelici.com](https://samuelefelici.com)

Built with React, Tailwind CSS, and Vite. Deployed automatically to GitHub Pages via GitHub Actions on every push to `main`.

## Local development

```bash
npm install
npm run dev
```

## Project structure

- `client/src/components/sections/` — page sections (Hero, Services, etc.)
- `client/public/assets/` — static assets (images, CV PDF)
- `client/src/index.css` — global theme / Tailwind config
